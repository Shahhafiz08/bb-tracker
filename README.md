# Bot-Bouncer Tracker

This Chrome Extension detects when you visit a subreddit or a post within a subreddit, and checks if `bot-bouncer` is listed as a moderator. If so, it shows a toast notification on your screen.

[![Download Extension](https://img.shields.io/badge/Download-Extension%20(.zip)-success?style=for-the-badge&logo=googlechrome)](https://github.com/Shahhafiz08/bb-tracker/archive/refs/heads/main.zip)

## How to Install

1. Click the download button above and extract the downloaded `.zip` file to a folder on your computer.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click **"Load unpacked"**.
5. Select the extracted folder (`bb-tracker-main`).
